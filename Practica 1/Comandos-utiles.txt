Algunos comandos útiles de bash:
- man,  Manual: -k <palabra clave>
- cd -Change directory: cd .. | cd / | cd ~
- ls –list directory contents: -l (detalles) | -a (ficheros ocultos)
- find –walk a file hierarchy: Sirve para buscar ficheros
- locate –find files by name
- cat –concatenate and print files:Imprime por pantalla el contenido de un fichero.
- head –print first lines: Imprime por pantalla las primeras líneas de un fichero (5 por defecto)
- tail –print last lines: Como head, pero con las últimas líneas
- less –visualizador  de  archivos: Permite  navegar  por  un  fichero
- top –display dynamic information about processes: Muestra de forma dinámica información sobre los procesos en ejecución
- ps –Process status: Muestra información sobre los procesos que son controlados por una terminal. -a (informacion de procesos de otros usuarios) | -l (informacion detallada)
- pstree –show the running process as a tree: Muestra todos los procesos siguiendo una estructura de árbol
- df –display free disk space:Muestra la cantidad de espacio libre en las distintas particiones de los discos duros. -h (human readable)
- '&' Se escribe al final de la línea a ejecutar. Provoca que el comando se ejecute en segundo plano, quedando la terminal libre para nuevas interacciones.
